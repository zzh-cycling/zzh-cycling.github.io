十月总结_那些可言不可言

十月留下了很多美好不美好的回忆。最想说的就是又是一段无疾而终的crush。

玩

烧烤滑雪

十月包括这个学期之后也就只有这三天假期，北航为了疫情防控把国情的七天假期缩短到三天几乎和没放假一样，本来该有的九天假期直接寄。我这三天过得也算比较充实吧，之前老早就和群友说要张罗一次野外露营烧烤，于是我拉了个航卫兵户外防火群筹备相关事宜并且水群，本来是想靠这个机会拉进和学妹的距离。结果十月一那天忙得脚不沾地，和大伙一起购物准备物资，中午接人送到露营地点，下午刚要开始做烧烤就被抓到了户外生火于是又骑车开始找可以生火的地方。好不容易找到一个地方就开始准备烧烤，然后拌饵料开始调漂钓鱼。最后烧烤也没吃上几口，淘宝买的五元神竿铅坠几乎没有重量，鱼也没钓到。

最后我喜欢的学妹还跟另一个一起烧烤的男生跑了，而那个男生本来是她俩之前在群里聊过我拉来当僚机的，最后我tm成局外人了。写下这段文字的我虽然已经过去了很长时间心还是痛的。虽然如此，但是。那一天的操劳确实很有趣，被抓之后货拉拉转移阵地也是很有意思，我在河边独钓满江风雨，旁有弄笛之音也未尝不惬意。十月二号报了雪协的乔波室内雪场活动，开始学起滑单板。感觉单板还是很帅，就是我刚上板时不会扣绑带穿错位置了结果人家只能剪掉扣了我100，就当买教训了。然后我开始滑，没想到第一次的我在教练wtq的教导下比较小心翼翼推坡还是直接摔。第二次没人看着我我直接速度加快起飞，刹车落叶飘直接就无师自通，开始练习走刃换刃。滑雪真好玩啊，不过可能室内乔波的雪比较粉，走不了刃，下次换室外的雪试试，感觉自己瞎琢磨出来的动作可能不规范不过将就着滑吧）。

野鸭湖

后来的10.23我又和学姐拉上了lpy和lyt重征野鸭湖，早上先赶上一波银杏林，很欣慰终于在本科阶段赶上了银杏林，大一大二的银杏林都因为疫情寄了，银杏林里人们零零散散已经有了很多，还有卖烤地瓜和零食、风筝的大叔，陆陆续续的法大央财的车协也到了，他们的小姐姐蛮好看的。于是我们出发的时间一推再推到了十一点，货拉拉去也。中午在一个王家鱼窝棚吃了铁锅炖大鱼，一般般吧，饼子不错，就是除了鱼以外的别的菜量都极少。这时窗外挂着呼呼的大风。学姐担心起了风，我看了一下天气，“没事，西风，回去爬坡省劲”。这时我才意识到这个回答是常识性问题，秋天可不就刮西风嘛。西风残照，汉家陵阙；西风袅袅秋；尽西风，季鹰归未？此时吃的这条松花鱼竟是如此的应景，无怪乎此时应该吃上一口鲈鱼美，鲈鱼正美不归去，空戴南冠学楚囚。要是那时燕地以及东北都在领土内，相比诗句也不只会让鲈鱼独美了。下午出发向野鸭湖，差不多两点正式开始了我们的野鸭湖征程，这个时间选的很好，阳光正好，微风不燥，很出图，只要不逆光拍就行，我在南方玩的那些图好看的基本都是这时候出来的）。

感觉这是我这几年来看过的最美丽的秋景，微波轻荡的湖面上，虽已经快入寒冬，仍有几只鸭子凫在水面上，大片大片的金黄色的芦苇丛啊，时有几张年轻的带笑的脸在苇叶中隐没浮现。我们沿着岸边走着，手里的相机自是没有停过，咔嚓，咔嚓，是蓝天碧水金色的芦苇帅气lpy和可爱学姐（），我想起来今日正好是霜降。那寒气凝集的湖水，以及身旁的芦苇丛，不由得让人想到两千年前同样的一幕——蒹葭苍苍，白露为霜，所谓伊人，在水一方。。绕着湖边约莫一圈，我们远远望见了官厅水库，北京最大的水表面积和蓄水量的水库，主要水流为河北怀来-永定河，还在夕阳的金色余辉中瞥见了观测塔，野鸭湖经典取景地。后来这一幕被我记录了下来，很多同学追问。

随后便是我们狼狈的往回赶的路程，四点出发，本以为六点能放完坡，七点能到沙河，妹想到立冬了，天黑的越来越早了，是时候猫在家里了，只有我们几个傻子出来闲逛，最后爬完返程的700+爬升后，到了山顶，我们看到了夕阳的最后一抹余辉，面前是一条条山谷陷出来的坳沟，在震撼中，享受完zyq学姐的零食小课堂后，我们不得不面对一个苦涩的事实：得摸黑放坡了。

抹黑放坡是紧张刺激的，但又令人恐惧，尤其是zyq学姐不小心撞到了手之后，我在lpy后面蹭灯放坡的同时一直频频回首望，所幸总能一直在一篇昏暗的国道旁看见两盏坚定的灯走了，于是继续向前走去，此时只有国道上的万里长风从西吹来，与我们同行的是满天繁星，隐约可以见到东南天空上的猎户星座。沉默着放完此段坡我们松了一口，最后决定坐地铁润回去了）。

野鸭湖、银杏林、大觉寺、以及其他的其他的北京的秋

百里画廊

还有就是看到了pyq里的图片开始畅想组织的百里画廊，虽然经历了很多波折出了很多意外最后还是蛮开心的。总结就是组织活动不要太多人，尽量还是找熟悉的小部分人彼此知晓能力的，这样时间一般能按照行程走。如果要是有很多人就一定要留足再留足机动时间，并且准备好plan B。感觉自己时间规划能力还是太弱了）

八点四十集合货拉拉去百里画廊，结果不出所料四个小组都推迟了，虽然最后我们组按时十点半到达，但是等16个人齐全了时间还是到了11.20. 然后前半程还是很不错的，大家骑得中规中矩玩得也蛮开心，景色也不错。
但是有大Pro带我们35+巡航50km 200爬升过早把自己拉爆 以至于后面爬坡没劲。最后吃饭的时间推迟到了1.30，等大家都到齐了又到了1.50，吃完出发又是2.30.后半程本来是我规划的很好的 享受之旅，一边欣赏景色一边慢悠悠爬坡，然后一路畅快放坡到黄花城水长城，长城埋瘞于水中，景色一绝，又可见其如龙之势起伏于山中。此时本应该是四点，正夕阳斜照，西风滋味。

然而意外来了，比明天先来。比如有一个学妹当时一直要去我给带上了，最后还是一直落在后面，使得规划的行程时间一直推，最后不得不又摸黑骑车。比如我自己感冒没好➕只睡了5h心率全程190+。爆掉了。比如我的腿从78km到88km腿一直抽筋嗯骑了10km，吃了一袋榨菜才好多了。最后患上了Ca不足恐惧症，吃了好几天钙片和菠菜，回来了后买了一箱奶nnd怎么四五天还没到。比如有三个人摔车。wtq更是整个人撞到了山壁上，就和阿拉菲利普一样，感觉好像是线碟热衰。比如最终也没在天黑之前出四季花海，我们几个人又拉成车灯小火车摸黑放坡。我还好穿的比较厚，有的同学真就只穿了个秋季骑行服我忽略了这一点，边骑边瑟瑟发抖。最终有好心的大娘大叔请我们喝羊汤，然后还开货车把我们拉了回来。

幸运的是最后大家安全的八仙过海各显神通都回来了，除了wtq受伤的世界达成了。）

生活

我当了量子力学课代表，很巧的是量子力学的助教正好是我组里大师姐，说到这不得不说的就是电动力学的助教正好之前迎新的时候认识了并且聊得很愉快，热统的助教也正好是关系很好的一个学长，理力的两个助教学姐倒是不是很熟悉，这不直接起飞（樂）。每周一次固定交作业收作业，和师姐谈一谈最近的科研进展，然后偶尔就会闲扯一些生活上的东西，是我少有的治愈时光，手机放一旁每次一谈就是一两个小时思维的肆意漫游很轻松自在。师姐很有亲和力，和师姐聊天总是很愉快。她在苦恼研二了导师还不给方向，每次组会就让随便讲讲最近看了啥，弄一个journal club氛围，我帮忙在一些物理群里问了问，收获了还不错的答案，并且也充实了我的升学文件夹里的CMTer指南。

学习

越发感觉到量子力学的重要性，而我之前只是学完了狄拉克形式理论部分，应用基本没看就去看量子场论和广相，注定对物理图像不会有深刻的认识。结合着近代物理实验对于物理系统终于有了更深刻的认识。比如很多东西都应该是经典量子对应起来来看，比如NMR的推导用到的旋转坐标系的方法我以前以为是相互作用表象，结果就是经典力学的一个很普通的做法，我也对Rabi oscillation有了更深层次的认识，谁能想象得到一个进动，一个交变磁场竟然和两能级系统的跃迁有关联呢。再比如平方反比有心力场SO(4)对称性的破缺，LRL矢量的经典结果和量子结果的对比。场论中对称性和场的变换、场的传播子散射振幅和统计物理中的配分函数的对应（即wick rotation），感觉逐渐把之前的零散的知识点都串了起来。

这个月学了很多东西，比如对量子力学的对称性终于有了点认识，群表示和量子力学结合起来才是最完美的，比如听了文小刚老师他们在2020年的拓扑序暑校对自发对称性破缺和拓扑序有了认识，也尝试着想了想简并的除了SSB的新机制，比如终于会蛙泳飘一会和踩水了，比如对二次量子化写哈密顿量，比如紧束缚模型、toric code model、ssh model,很多model（这个是wzh来我宿舍举行的一个小seminar，一起思考了色散关系的物理意义，和紧束缚模型的突破，感觉这种讨论形式很棒）。总的来说，收获很多，不会的仍有很多，还需要整理暑研、听力、物理……加油！

把用的vscode好好地配置了一番，加了Markdown All in One,Markdown Math, Markdown PDF,markdown toc后VSCODE的功能强大了很多，有很多快捷键，写起来更加享受了。

  • 提供了常用操作便利的快捷键
    • Ctrl B Bold
    • Ctrl I italy
    • Alt S 删除线
    • Ctrl shift ] uplevel
    • Ctrl shift [ downlevel
    • Ctrl M toggle math environment
  • 支持目录
    • Ctrl shift p ctoc 生成目录
    • Ctrl shift p utoc updating
    • 而且ctrl s 还可以更新
  • 一边书写一边预览(Ctrl + Shift + V or Ctrl + K V)
  • 可轻松转换为HTML文件和PDF文件
  • 优化了List editing的编辑 (Tab键 回车键都可以调整分级)
  • 可格式化table (Alt + Shift + F) 以及Task list (use Alt + C to check/uncheck a list item)，并且右边预览界面check code updating to time.
  • 支持特殊数学符号渲染 toggle
  • 而且实测引用图片什么甚至可以自动读取本地文件夹路径了，
